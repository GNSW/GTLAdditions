# 时空扭曲者

<BlockImage id = "time_space_distorter" scale = "8"/>

> 耗能倍数为0.1 \
> 拥有[复合配方类型](../multi_type.md): 量子操纵者, 深度化学扭曲仪 \
> 当运行**深度化学扭曲仪配方**时, 并行数计算公式为: 机器并行数/配方需求温度^0.8 \
> 配方的概率输入的概率变为原来的十分之一, 配方概率输入的电压加成均变为0. 若概率消耗的物品为**纳米蜂群**, 则变成**不消耗** \
> 机器成型后, 可以打开gui调整超频次数 \
> **超频次数1**: 额外消耗配方并行数/70mb的<FluidLink id="gtceu:infinity" />, 配方输出概率变成百分百 \
> **超频次数2**: 在超频1的基础上, 额外消耗配方并行数/180mb的<FluidLink id="gtceu:hypogen" />, 配方输出变成1.5倍 \
> **超频次数3**: 在超频2的基础上, 额外消耗配方并行数/340mb的<FluidLink id="gtceu:spacetime" />, 配方输出变成2.3倍 \
> 可以在gui中选择是否开启均分模式 \
> 均分模式**不受**超频次数调整**影响** \
> 开启后, 每个处理的配方都需要消耗配方并行数/53的<ItemLink id="kubejs:quantum_anomaly" />和配方并行数/873的<ItemLink id="kubejs:hypercube" /> \
> 将配方输出概率变成百分百, 同时配方输出变成1.7倍 \
> 由于奇迹内禀的不可预测性, 下面的配方概率无法被修改
