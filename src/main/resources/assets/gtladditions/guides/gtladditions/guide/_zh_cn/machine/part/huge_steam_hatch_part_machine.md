# 巨型蒸汽输入仓

<BlockImage id="huge_steam_input_hatch" scale="8" />

* 你可以在<Color color="#AA00AA">**EV**</Color>阶段进行制作 ~~(EV阶段末也是EV阶段)~~
* **巨型蒸汽输入仓**只能安装在多方块蒸汽结构上，替代**蒸汽输入仓**或**大型蒸汽输入仓**
* 它能将多方块蒸汽结构的处理配方电压等级提升到<Color color="#FFAA00">**HV**</Color>级别 (**你可以安装在大型蒸汽研磨机上来获得副产**)
* 它可以将所有处理配方的耗时变成1t，但是会消耗更多的蒸汽 (**请注意你的蒸汽库存!!!**)
