# 生物数据模拟室

<BlockImage id = "biological_simulation_laboratory" scale = "8"/>

* 只能安装<Color color="#FFFF00">**UXV**</Color>等级以上的激光仓

> 机器初始并行数为64, 可以在主机中放入对应的纳米蜂群获得额外的加成 \
> 放入<Color color="#CBCFD7">**铼纳米蜂群**</Color>时, **并行数**为2048, **耗时倍数**加成为0.9, **耗能倍数**加成为0.9 \
> 放入<Color color="#FF78C9">**山铜纳米蜂群**</Color>时, **并行数**为16384, **耗时倍数**加成为0.6, **耗能倍数**加成为0.8 \
> 放入<Color color="#FF77FF">**魔金纳米蜂群**</Color>时, **并行数**为262144, **耗时倍数**加成为0.4, **耗能倍数**加成为0.6 \
> 放入<Color color="#FF00FF">**不再是菜鸟的证明**</Color>时, **并行数**为4194304, **耗时倍数**加成为0.1, **耗能倍数**加成为0.25 \
> 放入<Color color="#FF00FF">**不再是菜鸟的证明**</Color>时不仅可以获得以上加成, 还能使机器可以以跨配方处理的方式工作 \
> 放入<Color color="#FF00FF">**不再是菜鸟的证明**</Color>的同时还解锁了配方输入带有**寰宇支配之剑**的配方 \
> 可以在主机gui界面中选择是否以跨配方处理的方式运行 \
> 该机器的跨配方模式只能选择**均分模式**, 不能选择**极限模式**
