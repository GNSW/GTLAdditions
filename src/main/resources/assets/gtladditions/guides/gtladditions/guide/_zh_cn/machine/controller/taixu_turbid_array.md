# 太虚浊化阵

<BlockImage id = "taixu_turbid_array" scale = "8"/>

> 可以打开主机的gui界面, 在gui界面的右下角物品槽放入对应的纳米蜂群或创造能源仓 \
> 当放入**创造能源仓**时, 放入一个便可获得3^16的并行和100%的成功率加成(~~放入多个效果相同~~) \
> 放入对应的纳米蜂群时, 获得额外的成功率加成, 加成数值为放入的纳米蜂群数量×基础加成 \
> 最多可以放入64个纳米蜂群或创造能源仓 \
> 放入**末影纳米蜂群**时基础加成为0.01 \
> 放入**龙纳米蜂群**时基础加成为0.05 \
> 放入**时空纳米蜂群**时基础加成为0.1 \
> 放入**永恒纳米蜂群**时基础加成为0.2

* 该机器结构只能放一个激光仓
* 当机器电压等级大于等于<Color color="#FFFF00">**UXV**</Color>时解锁**UU增幅液**输出
* 当机器电压等级大于等于<Color color="#FF0000">**MAX**</Color>时解锁**UU物质**输出
* ~~机器电压等级未达到时不会有额外的UU增幅液或UU物质输出~~
* 机器工作固定耗时5秒, 固定耗能为524288倍当前机器电压等级对应功率

> **~~大量公式警告~~**

* 每级恒星热力容器等级提供的加成α计算公式:
> <Latex math = "\alpha = 8 * (2^{恒星热力容器等级} - 1) * \sqrt{电压等级 + 1}" />

* 每级线圈提供的加成β计算公式:
> <Latex math = "\beta = 3.8 * 1.3^{线圈等级} * (\frac{线圈温度}{36000})^{0.7}" />

* uu增幅液成功概率计算公式:
> <Latex math = "1/(1 + e^{-0.1 * (\frac{\alpha}{50} + \frac{\beta}{100} + \frac{高度}{9})})" />

* uu增幅液的基础产出计算公式:
> <Latex math = "4096 * (1 - e^{-0.015 * (\alpha * \frac{高度}{16} + \beta * \ln{(电压等级 + 2)})})" />

* uu物质成功概率计算公式:
> <Latex math = "1/(1 - e^{-0.02 * (\frac{\alpha + \beta}{20} + \sqrt[3]{高度} * \frac{电压等级}{7})})" />

* uu物质的基础产出计算公式:
> <Latex math = "2250 * \tanh{(\sqrt{\alpha * \beta} * \frac{(高度 + 电压等级) * 0.06}{200})}" />

* 机器并行计算公式:
> <Latex math = "4096 * 1.621^{\min(\frac{线圈温度}{6400},12)}" />

* **注**: 公式中的高度指的是结构中重复层的高度, 类似于蒸馏塔或中子活化器的高度
