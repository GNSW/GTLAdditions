# Draconic Collapse Core

<BlockImage id = "draconic_collapse_core" scale = "8"/>

* You can put the **Harmonizing Core** in the machine, After placement, the output of the machine can be replaced with the ME Pattern Buffer
* Maximum parallel number x n for each voltage tier above <Color color="#00FF00">**UEV**</Color>
* When the **Harmonizing Core** is inserted, n is 12; when it is not inserted, n is 8.
* Can use large <Color color="#555555">**ULV**</Color> input bus to increase input capacity
