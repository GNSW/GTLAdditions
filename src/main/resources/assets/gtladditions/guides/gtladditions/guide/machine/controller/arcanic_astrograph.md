# Arcanic Astrograph

<BlockImage id = "arcanic_astrograph" scale = "8"/>

* It can resonate with all the **Harmony of Eyes** and **Arcanic Astrograph** within the current dimension, obtaining a parallel number bonus
* The maximum parallel number is 924 * ln(x), where x = a + 15b
* Here, a represents the number of **Harmony of Eyes** currently running, and b represents the number of **Arcanic Astrograph** currently running
* Only the demand for quantum chromodynamics bombs and cosmic elements varies with the parallel number. The power consumption and hydrogen-helium consumption are consistent with those of the **Harmony of Eyes**
