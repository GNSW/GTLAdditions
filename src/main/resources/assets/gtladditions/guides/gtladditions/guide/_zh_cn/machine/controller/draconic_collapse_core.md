# 龙怒坍缩核心

<BlockImage id = "draconic_collapse_core" scale = "8"/>

* 可以在主机中放入 <ItemLink id="harmonizing_core" />, 放入后可以将机器的输出**替换**为样板总成
* 电压等级每高出<Color color="#00FF00">**UEV**</Color>一级最大并行数Xn
* 当放入**调和核心**时, n为12; 未放入时n为8
* 可以使用巨型<Color color="#555555">**ULV**</Color>输入总线来提高输入容量
