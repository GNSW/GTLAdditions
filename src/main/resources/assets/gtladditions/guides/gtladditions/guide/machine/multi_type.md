# Multi Recipe Types

* When a machine allows processing multi recipe types, all recipes within each multi recipe type can be processed without modifying the recipe type
* For example: If a multi recipe type includes Pyrolyzer and Cracker, after selecting this multi recipe type, all recipes from both recipe types can be processed without changing the machine's recipe type
* ~~(Cannot guarantee that all multi recipe types won't have recipe conflicts)~~
* ~~"You can think of it as enhanced parallel processing across modes"~~
