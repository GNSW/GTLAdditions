# Transmutation Bus Hatch

<BlockImage id="gtladditions:me_block_conservation" scale="4" />

* Requires an **AE Item Storage Cell** to be inserted. All transmuted output items are first input into the cell and then returned to the AE network(Cannot fit into a **Portable Cell**)
* Hold <ItemLink id="gtceu:creative_data_access_hatch" /> and right-click the **Transmutation Bus** to use it in the **Door of Creation** and **Creative Aggregator** (will **consume** one Creative Data Access Hatch)
* Other usage methods are similar to the **ME Input Bus**. Mark the input items and quantities to automatically pull corresponding items from the AE network
