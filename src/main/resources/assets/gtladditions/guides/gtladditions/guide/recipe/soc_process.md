# Soc Production Line

> In GTLAdditions, a new production line has been added to complete the original Soc circuit levels \
> Soc circuits refer to: circuits synthesized using Soc chips to simplify the circuit assembly machine, which in most cases significantly reduces the steps and materials for circuit synthesis

<Row>
  <ItemImage id="gtladditions:echo_shard_boule" scale="4" />

  <ItemImage id="gtladditions:hassium_boule" scale="4" />

  <ItemImage id="gtladditions:starmetal_boule" scale="4" />

  <ItemImage id="gtladditions:periodicium_boule" scale="4" />

  <ItemImage id="gtladditions:infinity_boule" scale="4" />
</Row>

* Before making these boules, you need a new compound <ItemLink id="gtladditions:gallium_oxide_dust" />
* Use this compound, add **Boule** and corresponding dust, then process it in an Electric Blast Furnace with krypton to obtain the corresponding boule
