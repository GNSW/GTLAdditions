# 超级输入总成

<BlockImage id="super_input_dual_hatch" scale="8" />

* 你需要在<Color color="#00AA00">**UIV**</Color>阶段才能制作它
* 超级输入总成拥有大量的输入槽位, 可以容纳比巨型输入总成更多的物品和流体
* 拥有37个物品槽, 每个物品槽可以容纳2147483647(2.14G)个物品
* 拥有24个流体槽, 每个流体槽可以容纳2251799813685247mb(2.2TB)的流体
* 但是它有一个致命缺点: 每次返回物品或流体最多只能返回2.14G物品或2.14MB流体, 这导致如果你输入过多的流体在仓室内, 你需要多次点击返回才能将所有物品和流体返回
