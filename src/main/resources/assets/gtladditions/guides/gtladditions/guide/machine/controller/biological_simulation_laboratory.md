# Biological Simulation Laboratory

<BlockImage id = "biological_simulation_laboratory" scale = "8"/>

* Only laser hatches of <Color color="#FFFF00">**UXV**</Color> tier or higher can be installed

> The machine has an initial parallel count of 64, and can receive additional bonuses by placing corresponding nano swarms in the main machine \
> When placing <Color color="#CBCFD7">**Rhenium Nano Swarm**</Color>, **parallel count** is 2048, **time multiplier** bonus is 0.9, **energy multiplier** bonus is 0.9 \
> When placing <Color color="#FF78C9">**Adamantite Nano Swarm**</Color>, **parallel count** is 16384, **time multiplier** bonus is 0.6, **energy multiplier** bonus is 0.8 \
> When placing <Color color="#FF77FF">**Infinity Nano Swarm**</Color>, **parallel count** is 262144, **time multiplier** bonus is 0.4, **energy multiplier** bonus is 0.6 \
> When placing <Color color="#FF00FF">**Nan Certificate**</Color>, **parallel count** is 4194304, **time multiplier** bonus is 0.1, **energy multiplier** bonus is 0.25 \
> When placing <Color color="#FF00FF">**Nan Certificate**</Color>, not only do you get the above bonuses, but it also enables the machine to work in multi-recipe processing mode \
> When placing <Color color="#FF00FF">**Nan Certificate**</Color>, it also unlocks recipes that require **Infinity Sword** as input \
> You can choose whether to run in multi-recipe processing mode in the main machine's GUI interface \
> The machine's multi-recipe mode can only select **Equally Divided Mode**, not **Extreme Mode**
