# GTLAdditions机器配置

## **均分模式**

> 均分模式下, 机器会尽可能的处理全部能处理的配方, 配方并行数尽可能平均分配 \
> 该模式下, 总并行数为: 线程数 × 机器并行数

## **极限模式**

> 极限模式下, 机器会尽可能的集中处理一部分配方, 拥有无损超频和1toc(有关无损超频和1toc的概念请参考任务书) \
> 该模式下**可配置维护仓**的加成不会生效 \
> 该模式下, 线程数为最大可处理配方数

* **可以在模组配置中选择默认模式**, 初始默认模式为**均分模式**

#### **配方最短耗时配置**

> 可以在部分GTLAdd的机器gui左下角处打开按钮来调节 \
> 初始默认值为1s. 最低为5t, 最高为10s \
> 可以在模组配置文件中修改默认值
