# 物料谱解析仓

~~别名(op仓)~~

<BlockImage id="spectral_analysis_hatch" scale="8" />

* 需要在<Color color="#00AAAA">**UV**</Color>阶段才能制作它
* 物料谱解析仓只能安装在集成矿石处理厂或进阶集成矿石处理厂上，属于额外的附属仓室，只能安装一个物料谱解析仓到结构中
* 打开仓室gui可以看到有三个频段, 每个频段后面都有三个显示灯
* 每个频段的频号都是随机生成的 ~~(但是具有一定规律)~~
* 频段后的显示灯激活数量代表对应的激活等级, <Color color="#00FF00">绿色为激活状态</Color>, <Color color="#FF0000">红色则是未激活状态</Color>
* 激活等级1(数值范围 **±80**), 激活等级2(数值范围 **±35**), 激活等级3(数值范围 **±5**)
* 可以通过修改数值或通过拖拉文本框下的滑动条来调节频号


* 频段1: 线程加成, 直接应用到跨配方处理线程上
> **集成矿石处理厂**: 等级0(即默认加成): 4线程, 等级1: 6, 等级2: 8, 等级3: 10 \
> **进阶集成矿石处理厂**: 等级0(即默认加成): 72线程, 等级1: 96, 等级2: 128, 等级3: 144
* 频段2: 概率加成, 全部概率输出的电压增加概率**直接乘以**该频段等级数加一
> 例如: 在等级2的情况下, 输出物品A, 基础出产概率为30%, 基础电压加成概率为5%, 则经过该频段加成后基础电压加成概率为(5 * (2 + 1)) = 15%
* 频段3: 独立加成, 集成矿石处理厂与进阶集成矿石处理厂的加成不相同, 安装到不同机器结构中获得不同加成
> 安装到**集成矿石处理厂**时获得配方耗时倍率加成(**直接乘算**) \
> 安装到**进阶集成矿石处理厂**时获得流体消耗倍率加成(**直接乘算**) \
> 加成倍率: 等级0(即默认加成): 0.8, 等级1: 0.65, 等级2: 0.5, 等级3: 0.4

* 此外, 当输入频号与实际频号相同时(即零误差), 称为完美调频
* 当三个频段均为完美调频时, 机器超频从有损超频变成了无损超频而且可以使用极限处理模式来更高效地处理配方, 极大地提高了机器处理效率
