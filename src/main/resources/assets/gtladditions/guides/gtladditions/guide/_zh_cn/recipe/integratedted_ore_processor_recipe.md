# 集成矿石处理相关配方

<ItemLink id="gtceu:integrated_ore_processor" /> 配方

* ### 配方一
> 24号编程电路配方, 输入<ItemLink id="gtceu:jasper_ore" />, 可以直接得到<ItemLink id="gtceu:raw_tengam_dust" /> \
> 大大地简化了生镃粉的获取 \
> **但是**: 此配方只能在<ItemLink id="gtceu:advanced_integrated_ore_processor" /> 上运行!!!

* ### 配方二
> 8号编程电路和9号编程电路配方, 输入<ItemLink id="gtceu:cooperite_ore" />, <ItemLink id="gtceu:chalcocite_ore" />, <ItemLink id="gtceu:bornite_ore" />, <ItemLink id="gtceu:tetrahedrite_ore" />  \
> 可以直接得到对应的纯净矿石(即通过研磨机和洗矿机后得到的产物) \
> 稍微简化了铂系处理线的内容 \
> ~~输入粗矿也能得到对应的产物~~ \
> ~~(辉铜矿没有9号电路配方)~~
