# 奥术星图

<BlockImage id = "arcanic_astrograph" scale = "8"/>

* 可以与当前维度内的全部**鸿蒙之眼**与**奥术星图**形成共鸣,获得并行数加成
* 最大并行数为924 * ln(x),x = a + 15b
* 其中a为**正在运行**的鸿蒙之眼的数量, b为**正在运行**的奥术星图的数量
* **仅**量子色动力学爆弹和宇宙素的需求量随并行数变动,耗电与氢氦消耗均与鸿蒙之眼一致
