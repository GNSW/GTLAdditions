# 超维度化工厂

<BlockImage id = "dimensionally_transcendent_chemical_plant" scale = "8"/>

> 支持跨配方处理的化工厂, 拥有额外的耗时倍数和耗能倍数 \
> 耗时倍数: 0.8 * (1 - 线圈等级 * 0.05) \
> 耗能倍数: 0.6 * (1 - 线圈等级 * 0.05)
